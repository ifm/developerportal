:::{note}
When using multiple applications on a single device, keep in mind that the maximum number of simultaneous application instances is 20.
The maximum number of application of each type that can be configured is:
- 1 ODS application
- 2 PDS applications
- 1 PLC application
- 8 MCC applications
- 8 SCC applications

Multiple applications using the same camera cannot be active at the same time. Switch all the applications to CONF except the one currently running.
:::
