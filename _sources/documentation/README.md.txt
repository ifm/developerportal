# README

This project contains documentation for the O3R platform. It is used along with other projects to build to ifm3d.com site.
> Note: it does not cover any of the supported software stacks (ifm3d, ROS, etc).

Are you looking for ODS examples in python or c++? Go to the folder ODS/Python or ODS/Cpp.

Are you looking for utilities (calibration examples, 2D/3D registration, etc)? Go to the folder Software_Interfaces/Toolbox
